# Calculate Time Under pH Threshold

Calculates the total time each subject/group spends below a specified pH
threshold.

## Usage

``` r
calc_time_under_pH(x, ph_threshold = 5.4)
```

## Arguments

- x:

  Data frame containing pH data.

- ph_threshold:

  pH threshold (default is 5.4).

## Value

A data frame with the total time under the pH threshold for each
subject/group.
