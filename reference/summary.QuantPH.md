# Summarize Direct Estimation Results

Provides summary statistics for direct estimation results.

## Usage

``` r
# S3 method for class 'QuantPH'
summary(object, ...)
```

## Arguments

- object:

  Data frame containing direct estimation results with class 'QuantPH'.

## Value

A data frame with summary statistics for AUC, time under pH threshold,
minimum pH, and time to minimum pH.
